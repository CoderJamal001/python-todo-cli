# 📝 Python To-Do CLI App

This is a simple command-line To-Do application written in Python. It allows you to add, view, complete, and delete tasks.

## 🚀 Features

- Add new tasks
- View task list
- Mark tasks as completed
- Delete tasks

## 📦 How to Run

1. Make sure you have Python 3 installed
2. Run the script:
```bash
python todo.py
